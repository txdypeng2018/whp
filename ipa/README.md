# ipa打包 #

### 打包脚本 ###
- 执行机器需要先配置好开发者证书
- 在ios工程目录下执行[ipaBuild.sh]生成ipa安装包
- 执行前需要根据本机实际情况调整脚本内的路径
- 打测试包使用[AdHocExportOptions.plist]文件，打发布包使用[AppStoreExportOptions.plist]文件
